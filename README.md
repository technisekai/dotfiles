## Requirements
>Software yang dibutuhkan

Arch based
```
pacman -S openbox tint2 feh  picom obconf lxappearance 
```

Debian based
```
apt-get install openbox tint2 feh  picom obconf lxappearance 
```

## How
> Cara penggunaan

Pindah semua direktori ke home and enjoy

Shift + L = light theme

Shift + D = Dark theme
